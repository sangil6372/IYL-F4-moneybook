# 🪜💸 Vue.js-SADARI (가계부 프로젝트)

- Vue.js를 기반으로 한 가계부 관리 애플리케이션입니다.  
- 월별 수입/지출을 기록하고, 목표 설정과 통계를 통해 지출 습관을 파악할 수 있습니다.

## 🖥️ 프로젝트 소개

- **프로젝트명**: SADARI
- **주요 목표**: 직관적인 UI와 차트 기반 통계를 제공하는 사용자 친화적 가계부 앱 개발  
- **특징**: 캘린더 기반 거래 관리 + 목표 대비 소비 분석 + 회원 관리 기능 탑재

| 이름 | 역할 |

## 📌 주요 기능

### ✅ 로그인 / 회원가입

- 이메일 중복 체크
- 쿠키 기반 로그인 유지
- 주소 없이 간편 가입
- 회원 정보 저장 (user.json)

### 📋 대시보드

- 전체 거래 목록 확인
- 거래 추가, 수정, 삭제
- 카테고리/기간별 필터링

### 📆 캘린더

- 달력에서 수입/지출 확인
- 날짜별 거래 관리 (CRUD)
- 고정 거래 등록 가능

### 📊 분석

- 목표 금액 설정 및 변경
- 월별 소비 통계
- 날짜별 수입/지출 라인 그래프
- 도넛 차트로 비중 시각화

### 👤 프로필

- 유저 정보 확인 및 수정
- 프로필 이미지 업로드
- 목표 금액 변경 → 분석에 즉시 반영
- 로그아웃 및 회원탈퇴
